Sub Goal Tracker

A live, auto-updating sub goal overlay for OBS. Subs in, subs out — the bar always knows. Hit the goal and it celebrates, then rolls to the next one.

⚠️ One step left (site owner): this page needs a Twitch Client ID before the connect button works.
1. Register an app at dev.twitch.tv/console/apps (free)
2. OAuth Redirect URL: https://tezcreations.com/subgoal/ — Category: Broadcaster Suite — Client Type: Confidential (needed for the never-expiring token flow)
3. Paste the Client ID into CLIENT_ID at the top of the script, then set TWITCH_CLIENT_SECRET, SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY in Vercel, run subgoal/schema.sql in Supabase, and redeploy.
