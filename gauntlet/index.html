Add it in OBS as a Browser Source (width ~390, height ~600, transparent).

Chat commands (broadcaster + mods): !win <game>   !lost <game>   !active <game>. Use a name fragment or the list number, e.g. !win elden or !active 3.

First time? Run schema.sql in your Supabase SQL editor, then paste your project URL + anon key into config.js. That's the only setup.
